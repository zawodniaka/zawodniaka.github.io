Jet Set Go

Project Purpose

This project was done as a capstone project between a group of students with varying skill levels. More advanced students were to help develop the website, lead the other students, and creation of documentation. Jet Set Go is a fictional company that strives to make trip planning easier by being able to book hotels and flights from one website. Users were able to add and remove flights and hotels from their cart, browse flights and hotels that matched certain criteria (location and time), see their trips, and cancel trips. Admin users were able to see all registered users and their registered data, including their trips.

Project Stack

Node.js, Next.js, Supabase, Java, written in Visual Studio Code

Skills Used

Basic Java, basic HTML and JavaScript, React components, database creation, storing data, filtering data, and Tailwind CSS.

Book Stats- Console Version

Project Purpose

This project was done to showcase basic Java knowledge by creating a terminal interface to show some statistics read from a CSV file. The CSV file contains a list of books where each line has the title, author, release year, and a rating. The program utilizes a TreeMap to hold a set of keys that correspond to the release year of the books. Those keys are then used to store an ArrayList that contains all of the books for that year. Once all values are read in from the file, the program will start to list out each year and each book read that released in that year. Years are listed out in chronological order and books by their rating (in descending order).

Project Stack

Java, written in Visual Studio Code

Skills Used

Basic Java, file parsing, storing data using a map, sorting data
